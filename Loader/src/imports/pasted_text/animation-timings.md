Ajustement des timings

Je veux raccourcir et calibrer précisément la durée de chaque animation.

Respecter strictement les durées suivantes.

1. Loader (barre de chargement)

La barre doit se remplir de 0 % à 100 % en exactement 0,8 seconde.

Contraintes :

animation très fluide

progression continue

aucune pause

easing ease-in-out léger

style Apple

Durée totale :

0.8 seconde

2. Apparition du texte "Bienvenue"

Quand le loader disparaît :

Le texte “Bienvenue” apparaît.

Animation :

fade in

blur → net

apparition très smooth

Durée :

0.4 seconde

Easing :

ease-out

3. Disparition du texte

Le texte disparaît ensuite de manière fluide.

Animation :

fade out

léger blur

Durée :

0.4 seconde

4. Apparition du logo INFINIFY

Ensuite le logo INFINIFY apparaît.

Animation :

blur → net

opacity 0 → 100

léger scale 0.97 → 1

Durée :

0.4 seconde

Easing :

ease-out

5. Disparition du logo

Le logo disparaît ensuite.

Animation :

fade out

blur progressif

Durée :

0.4 seconde

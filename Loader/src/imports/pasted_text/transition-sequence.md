Transition après le loader

Quand le loader atteint 100 % :

La barre disparaît progressivement.

Utiliser une animation de blur + fade.

La disparition doit être très smooth, style Apple.

Durée : environ 0.6 à 0.8 seconde.

Le blur augmente pendant la disparition.

Effet attendu :

opacity diminue

blur augmente

l’élément se dissout doucement.

Apparition du texte

Après la disparition du loader :

Afficher le texte “Bienvenue” centré à l’écran.

Animation :

apparition smooth

blur → net

opacity → 100%

Timing :

durée apparition : 0.8 seconde

easing : ease-out

style très fluide.

Le texte doit rester visible 1.5 seconde.

Design :

texte blanc

typographie simple et élégante

parfaitement centré

style minimal Apple.

Transition vers le logo

Après les 1.5 secondes :

Le texte Bienvenue disparaît.

Animation :

fade out

léger blur

très smooth

durée : 0.6 seconde

Apparition du logo infini

Ensuite afficher le logo infini au centre.

Animation d’entrée :

blur → net

opacity → 100 %

très fluide

durée : 0.8 seconde

Option visuelle :

Ajouter un léger scale de 0.96 → 1 pour un effet premium.

Disparition du logo

Le logo reste visible environ 1 seconde puis disparaît.

Animation de sortie :

fade out

blur progressif

durée : 0.8 seconde

Contraintes importantes

toutes les animations doivent être très fluides

aucune animation brutale

style minimal / Apple-like

tout reste centré

fond sombre constant

transitions élégantes entre chaque étape.

Ordre final de la séquence

Loader se remplit

Loader disparaît (blur + fade)

Texte Bienvenue apparaît

Texte reste 1.5s

Texte disparaît

Logo infini apparaît

Logo reste ~1s

Logo disparaît
